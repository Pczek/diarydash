What?
This is diarydash, it's a minimalistic site to write a daily diary entry. Everyday you will be presented with a blank page with just the current date as a heading. Write whatever strives your mind, nobody will know and read. There is no need for fancy wording or nitpicking grammar and spelling. The main purpose is to whipe your mind's windshield.
How?
Every day you are presented with a blank page to fill. The entry is saved locally as you type using HTML5 local storage. You can access all of your entries via the menu at the bottom of the page. The file you will receive will be formated in markdown.
Why?
As most of us, I started out using a traditional paper diary, but it was quite annoying to remember writing an entry in the morning. Writing by hand also seemed way too slow, exhausting and wasteful with the resource. Next up I used tools like DayOne and shell scripts to write a daily entry, but it also turned out pretty unhandy. I finally realised I should do it in a Webbrowser, due to the fact I'am using it almost daily. So i created this page for simple input and assigned it as my first page when opening a new browser session and voila.
